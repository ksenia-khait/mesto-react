### Установка

Для запуска на локальной машине необходимо

1. Установить npm зависимости:

##### ` npm install `

2. Запустить в режиме разработки:

##### ` npm start `

Если все прошло успешно, проект будет запущен на

##### ` http://localhost:3000 `
